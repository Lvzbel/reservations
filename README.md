## Installation
Requirement: Docker

Step One:

```
docker run --rm \
    -u "$(id -u):$(id -g)" \
    -v "$(pwd):/var/www/html" \
    -w /var/www/html \
    laravelsail/php82-composer:latest \
    composer install --ignore-platform-reqs
```

Step Two: build container

```
./vendor/bin/sail build
```

Step Three:

```
./vendor/bin/sail up -d
```

Step Four: (seed db)

```
./vendor/bin/sail artisan migrate:fresh
```
